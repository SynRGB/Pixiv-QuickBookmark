## 简介

点击图片主区域，会直接收藏图片，而不是跳转到图片页面。并额外添加一个按钮用于跳转到图片页面。

### 更新说明

- `v2.4` 进一步优化运行效率
- `v2.2` 进一步优化运行效率
- `v2` 解决了作品页面点击跳转按钮总是强制跳转到新页面的问题，现在与原版相同是局部刷新
- `v2` 优化运行效率
